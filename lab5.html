Lab 5: BYO MXNet
Handwriting/Digit Identification

Step 1

Setup

In this next lab, we’ll use deep learning to train to recognize hand writing. The dataset being used is 70,000 grey scale hand written digits that have already been labeled for us.

We’ll use this dataset splitting it into a training and validation dataset (60,000 and 10,000 respectively).

The output of the neural network is 10 different classifications or labels. That’s because it’s identifying the numbers between 0-9 of a digit.

You’ll notice the script that does the actual work is in the same folder but separate from the actual python notebook itself. We’ll step through both of them during this lab.
