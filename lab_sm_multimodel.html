Step 2

Set IAM permission

We will build a container that fulfills all of the multi-model endpoint container requirements, and then upload that image to Amazon Elastic Container Registry (ECR). Because uploading the image to ECR may create a new ECR repository, this notebook requires permissions in addition to the regular SageMakerFullAccess permissions. The easiest way to add these permissions is simply to add the managed policy AmazonEC2ContainerRegistryFullAccess to the role that you used to start your notebook instance.

Please go back to SageMaker console, locate the notebook instance and click for detail.

You can find the IAM permission used for the notebook instance in the Permissions and encryption section. Please click on the IAM role ARN which will take you to IAM console.

Click Attach policies button. Search for AmazonEC2ContainerRegistryFullAccess, an AWS managed policy, and check the policy. Hit Attach policy button.
Now you have attached the AmazonEC2ContainerRegistryFullAccess policy to the notebook execution role. There's no need to restart your notebook instance when you do this, the new permissions will be available immediately.
